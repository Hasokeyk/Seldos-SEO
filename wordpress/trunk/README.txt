== Description ==

All seo settings can be made with this plugin. Google analytic, yandex metrica, Search console, title and desc, Visual seo, UX and UI edits


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `seldos-seo.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why should I use the plugin?
This plugin covers all WordPress seo settings.

= How do I add Google analytics?
You can adjust all the tools from the menu "Seldos SEO"> "Settings"> "General".
